# Data Center Fiber Optics Full Exam

A self-contained 130-question interactive exam based on the supplied
Data Center Fiber Optics & Network Infrastructure training manual.

## Publish with GitHub Pages

1. Create a new public GitHub repository.
2. Upload `index.html` and `.nojekyll` to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select **main** and **/(root)**, then save.

The public address will be:

`https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/`
